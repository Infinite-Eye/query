# Installation

Add dependencies to composer.json file, and install with composer.json

```
{
  "repositories": [
    {
      "type": "vcs",
      "url": "https://github.com/Infinite-Eye/loop.git"
    }
  ],
  "require": {
    "infinite-eye/loop": "dev-main"
  }
}
```

# Usage

```
\InfiniteEye\Query\Loop::content();
```
